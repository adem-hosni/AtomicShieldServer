# AtomicShield Mock Data Server (Deprecated)

> ⚠️ **This server is deprecated and only serves mock data.**

This Express backend was used for development and prototyping. All endpoints return in-memory mock data — no real database, no real OAuth, no real anti-cheat integration.

- **Mock data only** — players, bans, dashboard stats, auth are all hardcoded/fake
- **Not production ready** — plaintext passwords, fallback secrets, OAuth stubs
- **Use the actual FiveM resource/integration** for real anti-cheat functionality
- **Kept for reference** — API contracts and endpoint structure documentation
